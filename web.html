JAVA BEANS:

Introduction to java beans:

A java bean is a java class that encapsulates many object, into a single object, this object is called a bean.The class of java bean is written in java language.
The java class that is there is reusable that means it can be used many times. Java Bean was developed by Sun Microsystem.
The Java Bean is a general-purpose component model.

Properties of Java Bean:

1.A java bean class has a no-arg constructor.
2.It uses getter and setter function.
3.Java Bean is Serializable.

Example of Java Bean:-

Example:

Advantages of Java Bean:-

1.The java bean follows the Properties of Write once Run anywhare.
2. Java Bean program can work in different local platflorm.
3.Java Bean is platflorm independent,it gets run everywhare.
4.You can easily configure java Bean.
5.The Properties,events,and methods of Java Bean can controlled.
6.Configuration setting in Java Bean can be saved in persistent storage and it can be restored later.
