MOBILE-COMPUTING

About Mobile-Computing:

the concept of wireless communication technologs and the proliferation of portable computing devices have made possible a mobile computing era in which users. on the move,can seamlrssly access network services and resources, from anywhare and at anytime .this course provides an Introduction to the fundamentals of mobile computing. A background in computer network and wireless communication is required.

Introduction to mobile-computing:

Mobile computing is a technology by which users can transmit data,voice,and video through a mobile or any other wireless device.
In other word ,"Mobile-Computing is a technology that is used to transmit data through a mobile device over a wireless network."

In general,mobile computing is seen as a technology used in smartphones,but it is a much bigger topic because it is not limited to smartphones, it is used in every device that supports mobility.Such as:-laptop,smart watch,tablet etc.

There are three main concepts of mobile computing on which it is based:-
  1. Mobile communication
  2. Mobile hardware
  3. Mobile software

Mobile communication:

Here mobile communication refers to an infrastructure that provides relible and hassle-free communication for wireless device.
It includes the protocols,services,bandwidth and portals that the devices support.
The Data format is also defined within this,ensuring that there is no collision with other systems.
Radio wave infrastructure is used in mobile, in which signals are sent over the air (ie the sending and receiving of signals is through the air. )

Mobile hardware:

Mobile hardware means those devices or mobile devices that can receive or access the service of mobilit.It include laptops,smartphones,tablet PCs,smart watches and other personal digital assistants (PDAs).
A receptor is installed inside these devices which is designed to sense and receive signals.And this device works on 'full duplex'i.e it can also send and receive signals at the same time
Mobile hardware works over a wireless network.

Mobile software:

Mobile software is program that runs on mobile hardware.It can also be called 'Operating system'of mobile.
It remains responsible for all the function inside the device.It is also called the engine of the device.
It looks after all the features and requirements of mobile application.It provides features like camera,music player,cellular connectivity, Wi-Fi,Bluetooth,voice recorder,speech recognition,and video player.
