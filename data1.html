Introduction to Data-Mining:

What is Data-Mining:

Data mining is also called data or knowledge discovery.Data mining is the process of searching small data from a group of very large data.Traditional statistics,artificial intelligence and computer graphics are used in this process.
In other word,"data mining is a useful technique by which companies extract important information from a group of data."
Using data mining ,hidden patterns and useful data are discovered and than decision making is made on the basis of these patterns and data by using the process of data mining,organization slove the problems in business.
data mining tools are used to analyze data in data mining.This tool is very powerful.

Goals of Data-mining:

1:-Explanatory:

In this, the event or situation seen is explained

2:-Confirmatory:

In this, hypotheses free from possibilities are Confirmed.

3:-Analyzatry:

In this new data is analyzed so that positive feedback can be given.
