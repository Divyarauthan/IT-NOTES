About Dot Net Programming:

VB .NET is the visual programming technique based on Object Oriented Concept.This subject will give the students an in-depth understandinng the features of VB .NET.The practical exercise of VB .NET during the course of study will reinforce the understandinng of the subject.

Introduction to Microsoft .Net Framework:

.NET is a Framework for developing software application.It is designed and developed by Microsoft and the first beta version was released in 2000.
It is used to devlop application for web,windows,phone.Apart from this,it offers a broad range of fuctionalities and support.
This frame work contains a large number of class libraries known as Framework class libraries(FCL).Software programs written in .NET are executed in an execution environment, which is called Common Language Runtime(CLR).These are the main and essential parts of the .NET Framework.
It provides various services like Framework,memory management,networking,security,and type-safety.
